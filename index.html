Key Terms

IP Address

A unique number that identifies every device on a network.

Packet

A small chunk of data sent over a network between devices.

Protocol

A set of rules that controls how data is sent and received.

Bandwidth

The maximum amount of data that can travel through a connection.

Router

A device that directs data packets between computer networks.

Firewall

A security system that monitors and controls incoming network traffic.
