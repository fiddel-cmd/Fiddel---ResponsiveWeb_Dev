Transmission Control Protocol (TCP)

Introduction

TCP is a core protocol of the Internet Protocol Suite that provides reliable, ordered, and error-checked delivery of data between applications running on hosts across an IP network.

How TCP Works

TCP establishes a connection between two endpoints using a three-way handshake (SYN, SYN-ACK, ACK). Once connected, TCP segments data, numbers packets, and retransmits lost packets to ensure reliable delivery.

Features

  • Connection-oriented communication
  • Reliable delivery with retransmission
  • Flow control (sliding window)
  • Congestion control

Use Cases

TCP is used by applications that require reliable transport such as web browsing (HTTP/HTTPS), email (SMTP), and file transfers.

Conclusion

TCP enables dependable communication on the internet. While it adds overhead for reliability, it's essential for applications where data integrity matters.
